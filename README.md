# Projeto Aulas

Uma formação aberta, em português, para estudar desenvolvimento fullstack JavaScript/TypeScript do zero até tópicos avançados de engenharia de software.

Este repositório nasce como apoio para uma série de aulas no YouTube. A ideia é simples: estudar ensinando. Cada aula deve transformar um assunto em um roteiro claro, prático e consultável, servindo tanto para quem acompanha a sequência completa quanto para quem chega procurando um tema específico.

## Proposta

O projeto tem três objetivos principais:

- Construir uma trilha sequencial do absoluto zero até uma atuação fullstack mais madura;
- Organizar o conteúdo em Markdown para estudo independente;
- Criar uma base reutilizável para produzir aulas, exercícios e prompts de estudo com IA.

## Como Usar

Se você está estudando, comece pelo [currículo](curriculo.md). Ele apresenta a ordem macro sugerida e mostra como os assuntos se conectam.

Se você quer entender como o repositório será organizado nas próximas versões, leia a [estrutura do repositório](estrutura-do-repositorio.md).

Se você quer contribuir, revisar ou produzir uma aula, veja:

- [Template de aula](templates/aula.md)
- [Template de módulo](templates/modulo.md)

## Status

Este projeto está na V2 de estrutura editorial. Nesta fase, o foco é manter a matriz curricular, os templates e as pastas de módulos preparados para receber as aulas.

As aulas e arquivos individuais de cada assunto serão criados em uma etapa posterior.

## Princípios

- Uma aula por assunto sempre que isso ajudar a consulta futura.
- Aulas gerais antes de assuntos muito específicos.
- Conteúdo direto, sem excesso de texto.
- Roteiros pensados para vídeo, mas úteis como material de estudo.
- Exercícios e prompts de IA ao final das aulas para reforçar o aprendizado.
- Sequência única do zero ao avançado, mesmo que o repositório seja modular.
