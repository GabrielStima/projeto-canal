# Exercício 03 — Dados, Storage e Entrega

## O que você já tem

Você definiu requisitos, rede e identidades para a infraestrutura cloud.

## Tarefa

Classifique cinco tipos de dados do sistema, incluindo:

- registros transacionais;
- uploads ou documentos;
- frontend estático;
- logs;
- backups.

Para cada tipo, escolha entre banco gerenciado, object storage, CDN ou outra opção já estudada. Registre:

1. motivo da escolha;
2. necessidade de criptografia e controle de acesso;
3. retenção e ciclo de vida;
4. backup ou possibilidade de reconstrução;
5. risco de cache desatualizado;
6. responsabilidade operacional da equipe.

Inclua uma decisão sobre réplica de leitura ou explique por que ela ainda não é necessária.

## O que você vai produzir

Uma matriz de dados e entrega que evita usar banco, filesystem e CDN como soluções intercambiáveis.
