# SAST

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar o que é SAST;
- entender que tipo de problema SAST costuma encontrar;
- diferenciar análise estática de teste em execução;
- usar resultados de SAST com contexto e priorização.

## Ideia Central

SAST significa Static Application Security Testing. É análise de segurança feita sobre código, configuração ou artefatos estáticos, sem precisar executar a aplicação como um usuário real.

SAST pode ajudar a encontrar padrões perigosos cedo: uso inseguro de entrada, query montada manualmente, segredo hardcoded, configuração frágil ou fluxo de dados suspeito.

Mas SAST não entende todo o contexto de negócio. Ele ajuda a apontar sinais. O time ainda precisa revisar impacto, explorabilidade e correção.

## Exemplo Prático

Um scanner sinaliza uma query SQL construída com concatenação de string.

Isso pode indicar risco de SQL injection. A investigação precisa olhar:

- a entrada vem de usuário?
- existe validação?
- o banco usa query parametrizada em outro ponto?
- o trecho roda em produção?
- a mudança é nova?

O valor do SAST está em encontrar cedo o lugar que merece revisão.

## Analogias e Intuições

- SAST é uma leitura técnica do código com uma lupa de segurança.
- Ele encontra cheiro de problema, mas nem todo cheiro é incêndio.

## Erros Comuns

- Achar que SAST prova que a aplicação é segura.
- Ignorar todos os alertas por causa de falsos positivos.
- Bloquear qualquer achado sem triagem.
- Rodar SAST só no fim do projeto.
- Não ensinar o time a interpretar os resultados.

## Checklist de Entendimento

Antes de seguir, a pessoa deve conseguir responder:

- O que SAST consegue analisar?
- O que SAST não consegue provar?
- Como priorizar um alerta?
- Qual é a diferença entre falso positivo e falso negativo?
- Como SAST entra no fluxo de pull request?

## Prática Recomendada

- Pegue um trecho de código com entrada de usuário.
- Liste padrões que um SAST poderia sinalizar.
- Classifique cada sinal como alto, médio ou baixo risco com justificativa.
