# Internet vs Web

## Objetivo

Ao final desta aula, você deve ser capaz de:

- diferenciar Internet e Web;
- explicar por que a Web depende da Internet;
- reconhecer exemplos de serviços que usam Internet sem serem Web;
- entender por que essa diferença importa para desenvolvimento fullstack.

## Ideia Central

Internet e Web não são a mesma coisa.

A Internet é a infraestrutura global que conecta computadores, celulares, servidores, roteadores e outros dispositivos. Ela permite que dados circulem entre pontos diferentes.

A Web é um dos serviços que rodam sobre essa infraestrutura. Ela usa navegadores, servidores, URLs, HTTP, HTML, CSS e JavaScript para entregar páginas, aplicações e conteúdos interativos.

Uma forma curta de pensar: a Internet é a estrada; a Web é um dos sistemas de entrega que usa essa estrada.

## Exemplo Prático

Compare estas situações:

- Abrir um site no navegador.
- Enviar uma mensagem por um aplicativo.
- Assistir a um vídeo por streaming.
- Fazer uma chamada de voz.
- Usar um jogo online.

Todas usam Internet, mas nem todas são Web no sentido clássico de páginas e aplicações acessadas por navegador.

## Analogias e Intuições

- A Internet é a infraestrutura de transporte; a Web é um serviço que trafega por ela.
- A Internet é como uma rede de ruas; a Web é como lojas, placas, entregas e vitrines usando essas ruas.
- A Internet conecta máquinas; a Web organiza documentos, aplicações e interações para pessoas.

## Erros Comuns

- Usar Internet e Web como sinônimos em qualquer contexto.
- Achar que tudo que usa Internet é uma página web.
- Pensar que desenvolvimento web começa apenas no HTML.
- Ignorar que aplicações web dependem de infraestrutura, endereçamento e comunicação.
