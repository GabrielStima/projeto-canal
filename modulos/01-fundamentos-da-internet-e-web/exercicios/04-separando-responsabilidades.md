# Exercício 04 — Separando Responsabilidades

Faça este exercício depois da aula [Frontend e Backend Como Papéis na Web](../01.09-frontend-e-backend-como-papeis-na-web.md).

## Cenário

No portal do tutor, uma tela mostra:

- nome do pet;
- status atual do atendimento;
- horário da última atualização;
- botão para atualizar as informações.

## Sua Tarefa

Classifique cada responsabilidade:

| Responsabilidade | Frontend, backend ou ambos? | Por quê? |
| --- | --- | --- |
| Mostrar o nome e o status na tela | | |
| Buscar o status armazenado | | |
| Decidir quais dados podem ser vistos pelo tutor | | |
| Enviar um novo pedido ao clicar no botão | | |
| Informar que ocorreu um erro | | |

Não existe apenas uma forma de responder a todos os itens. O importante é justificar onde cada responsabilidade começa e onde depende do outro lado.

## Corrija Sua Atividade Com IA

Depois de concluir a atividade, copie o prompt abaixo, substitua o campo indicado pela sua resposta e envie para uma IA:

```text
Estou estudando frontend e backend como papéis em uma aplicação web.

Resolvi uma atividade sobre uma tela que mostra o nome de um pet, o status do atendimento, o horário da última atualização e um botão para buscar informações atualizadas.

Minha tarefa foi classificar como frontend, backend ou ambos estas responsabilidades:
- mostrar o nome e o status na tela;
- buscar o status armazenado;
- decidir quais dados podem ser vistos pelo tutor;
- enviar um novo pedido ao clicar no botão;
- informar que ocorreu um erro.

Também justifiquei cada classificação. Algumas responsabilidades podem envolver os dois lados.

Analise minha resposta usando estes critérios:
- apresentação e processamento foram diferenciados;
- frontend não foi tratado como apenas aparência;
- backend não foi tratado como apenas banco de dados;
- responsabilidades compartilhadas consideram a cooperação entre os dois lados;
- as justificativas são coerentes com cada classificação.

Primeiro, aponte o que está correto. Depois, indique erros ou partes imprecisas sem reescrever toda a atividade por mim. Para cada problema, dê uma dica e permita que eu tente corrigir. Só apresente uma resposta completa se eu pedir.

Minha resposta:
```
