## Ideia Central

CSS define como o conteúdo HTML será apresentado. Com ele, controlamos cores, espaçamentos, fontes, bordas, tamanhos e muitas decisões visuais.

O ponto mais importante no começo é entender que CSS funciona por regras aplicadas a elementos. Essas regras controlam a apresentação e alteram o espaço ocupado pelos elementos na tela. A forma como regras competem e herdam valores será aprofundada na próxima aula.

## Exemplo Prático

```css
body {
  font-family: Arial, sans-serif;
  line-height: 1.5;
  margin: 0;
  color: #222;
  background: #f5f5f5;
}

main {
  max-width: 720px;
  margin: 0 auto;
  padding: 24px;
}

button {
  padding: 8px 12px;
  border: 1px solid #222;
  background: white;
}
```

## Analogias e Intuições

- HTML diz o que cada parte é; CSS diz como ela aparece.
- A cascata permite combinar regras de diferentes lugares.
- Box model é a área real que cada elemento ocupa na página.

## Erros Comuns

- Tentar resolver estrutura ruim apenas com CSS.
- Esquecer que regras podem sobrescrever outras regras.
- Confundir `margin` e `padding`.
- Usar seletores muito específicos cedo demais.
- Colocar todo estilo direto no atributo `style`.
