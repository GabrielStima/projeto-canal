# Layout com CSS

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar o que é layout em uma página web;
- reconhecer o fluxo normal dos elementos;
- usar Flexbox e Grid em exemplos simples;
- escolher uma estratégia de layout sem decorar todas as propriedades.

## Onde Esta Aula Entra na Formação

- Módulo: Frontend Essencial
- Aula anterior: Convenções de Nomenclatura
- Próxima aula: Responsividade

## Ideia Central

Layout é a organização dos elementos na tela. Antes de pensar em animações ou componentes, é preciso decidir como cabeçalho, navegação, conteúdo, colunas, cards, formulários e botões ocupam espaço.

CSS tem vários mecanismos de layout. Neste ponto da formação, o foco é entender fluxo normal, Flexbox para distribuição em uma dimensão e Grid para estruturas em linhas e colunas.

## Tópicos da Aula

- Fluxo normal do documento
- Elementos em bloco e inline em alto nível
- Espaçamento e alinhamento
- Flexbox para linhas ou colunas simples
- Grid para grades simples
- Quando evitar `position` como primeira solução

## Exemplo Prático

```css
.attendance-list {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

.attendance-card__actions {
  display: flex;
  gap: 8px;
  justify-content: flex-end;
  align-items: center;
}
```

## Analogias e Intuições

- Layout é arrumar os móveis antes de escolher a decoração.
- Flexbox é ótimo para organizar uma linha ou uma coluna.
- Grid é ótimo quando você pensa em linhas e colunas ao mesmo tempo.

## Erros Comuns

- Usar `position: absolute` para layout comum.
- Tentar alinhar tudo com margens manuais.
- Aplicar Flexbox ou Grid no elemento errado.
- Ignorar o tamanho do conteúdo real.
- Criar layout bonito em uma largura e quebrado em outra.

## Checklist de Entendimento

## Prática Recomendada

- Crie uma seção com uma lista de atendimentos do dia, mostrada como cards.
- Organize os cards com Grid.
- Crie uma área de botões dentro de cada card e alinhe com Flexbox.
- Mude a quantidade de conteúdo em um card e veja como o layout reage.
