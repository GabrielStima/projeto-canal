# Semântica HTML

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar o que é semântica HTML;
- escolher elementos pelo significado, não pela aparência;
- reconhecer por que semântica ajuda acessibilidade, manutenção e SEO;
- saber quando usar elementos genéricos.

## Ideia Central

Semântica HTML é usar elementos que descrevem o papel do conteúdo. Um título deve ser um título, uma navegação deve ser uma navegação, o conteúdo principal deve estar em uma área principal e um botão deve ser um botão.

Isso ajuda o browser, tecnologias assistivas, mecanismos de busca e outros desenvolvedores a entenderem a página. Semântica não é detalhe estético; é parte da qualidade da interface.

## Tópicos da Aula

- O que significa semântica
- Elementos semânticos comuns: `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`
- Hierarquia de títulos
- Botão vs link
- Elementos genéricos: `div` e `span`
- Relação entre semântica e acessibilidade

## Exemplo Prático

Compare uma estrutura genérica com uma estrutura mais significativa:

```html
<header>
  <h1>Canal de Estudos</h1>
  <nav>
    <a href="/aulas">Aulas</a>
    <a href="/projetos">Projetos</a>
  </nav>
</header>

<main>
  <section>
    <h2>Aulas recentes</h2>
    <article>
      <h3>HTML</h3>
      <p>Aprenda a estruturar uma página web.</p>
    </article>
  </section>
</main>
```

## Analogias e Intuições

- Semântica é colocar placas corretas nos cômodos de uma casa.
- `div` é uma caixa sem nome; útil, mas não deve substituir tudo.
- Um HTML bem semântico continua compreensível mesmo antes do CSS.

## Erros Comuns

- Usar `div` para tudo.
- Usar link para ação que deveria ser botão.
- Usar botão para navegação que deveria ser link.
- Escolher `h1`, `h2` e `h3` pelo tamanho padrão.
- Criar várias áreas principais em uma página simples sem necessidade.

## Checklist de Entendimento

Antes de seguir, você deve conseguir responder:

- O que é semântica HTML?
- Por que um botão não é igual a um link?
- Quando `div` faz sentido?

- Como semântica ajuda acessibilidade?
