# Gerenciamento de estado

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar o que é estado em uma interface;
- diferenciar estado local, estado compartilhado e estado derivado;
- decidir onde um estado deve morar em uma árvore de componentes;
- reconhecer quando uma biblioteca de estado pode ser exagero.

## Ideia Central

Estado é qualquer informação que muda ao longo do tempo e influencia a interface. Pode ser um texto digitado, um item selecionado, uma lista carregada ou uma aba ativa.

Gerenciar estado não significa instalar uma biblioteca. Antes disso, é preciso entender onde o dado nasce, quem precisa dele e se ele realmente precisa ser guardado.

Muitos problemas de estado são resolvidos com estado local, props, estado derivado e levantamento de estado para um componente pai.

## Exemplo Prático

```tsx
function FiltroAtendimentos() {
  const [busca, setBusca] = useState("");
  const atendimentosFiltrados = atendimentos.filter((atendimento) =>
    atendimento.pet.toLowerCase().includes(busca.toLowerCase())
  );

  return (
    <>
      <input value={busca} onChange={(evento) => setBusca(evento.target.value)} placeholder="Buscar pet..." />
      <ListaAtendimentos atendimentos={atendimentosFiltrados} />
    </>
  );
}
```

`busca` é estado. `atendimentosFiltrados` é derivado desse estado e não precisa ser guardado separadamente. No PetCare OS, a recepção usa esse padrão para filtrar a fila de atendimentos do dia por nome do pet ou status.

## Analogias e Intuições

- Estado deve ficar perto de quem usa, mas alto o suficiente para quem precisa compartilhar.
- Estado derivado e como um calculo: se você consegue recalcular, talvez não precise armazenar.

## Erros Comuns

- Guardar em estado algo que poderia ser calculado.
- Duplicar a mesma informação em vários lugares.
- Subir todo estado para o topo da aplicação sem necessidade.
- Instalar biblioteca antes de entender o problema.

## Checklist de Entendimento

## Prática Recomendada

- Crie uma lista filtrável com estado local.
- Identifique quais valores são estado e quais são derivados.
- Refatore um componente para mover estado para o pai quando dois filhos precisarem dele.
