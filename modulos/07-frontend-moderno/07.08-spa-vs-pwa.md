## Ideia Central

SPA significa Single Page Application. É um modelo em que uma página principal carrega JavaScript e a navegação entre telas acontece principalmente no cliente.

PWA significa Progressive Web App. Não é apenas "uma SPA melhor". PWA se refere a capacidades progressivas da Web, como instalabilidade, funcionamento offline parcial, service workers, cache controlado e experiência mais próxima de aplicativo.

Uma aplicação pode ser SPA sem ser PWA. Também pode usar recursos de PWA sem seguir exatamente o modelo mental de uma SPA tradicional.

## Exemplo Prático

Compare dois cenários:

```text
SPA:
- URL muda
- JavaScript troca a tela
- Dados podem ser buscados via API

PWA:
- Pode instalar no dispositivo
- Pode usar service worker
- Pode responder melhor a offline/cache
```

## Analogias e Intuições

- SPA descreve como a aplicação navega.
- PWA descreve capacidades extras que aproximam a Web de uma experiência de app.

## Erros Comuns

- Achar que toda SPA é automaticamente PWA.
- Prometer offline completo sem entender cache e dados.
- Ignorar SEO, acessibilidade e primeira carga.
- Confundir PWA com aplicativo nativo.
