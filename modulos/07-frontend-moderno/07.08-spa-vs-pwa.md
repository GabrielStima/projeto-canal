# SPA vs PWA

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar o que e uma SPA;
- explicar o que e uma PWA em alto nivel;
- diferenciar modelo de navegacao de capacidades progressivas;
- reconhecer quando cada termo aparece em projetos reais.

## Ideia Central

SPA significa Single Page Application. E um modelo em que uma pagina principal carrega JavaScript e a navegacao entre telas acontece principalmente no cliente.

PWA significa Progressive Web App. Nao e apenas "uma SPA melhor". PWA se refere a capacidades progressivas da Web, como instalabilidade, funcionamento offline parcial, service workers, cache controlado e experiencia mais proxima de aplicativo.

Uma aplicacao pode ser SPA sem ser PWA. Tambem pode usar recursos de PWA sem seguir exatamente o modelo mental de uma SPA tradicional.

## Exemplo Pratico

Compare dois cenarios:

```text
SPA:
- URL muda
- JavaScript troca a tela
- Dados podem ser buscados via API

PWA:
- Pode instalar no dispositivo
- Pode usar service worker
- Pode responder melhor a offline/cache
```

## Analogias e Intuicoes

- SPA descreve como a aplicacao navega.
- PWA descreve capacidades extras que aproximam a Web de uma experiencia de app.

## Erros Comuns

- Achar que toda SPA e automaticamente PWA.
- Prometer offline completo sem entender cache e dados.
- Ignorar SEO, acessibilidade e primeira carga.
- Confundir PWA com aplicativo nativo.
