# Error boundaries

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar por que erros precisam de UI propria;
- usar `error.tsx` no App Router;
- entender recuperacao de erro em uma rota;
- diferenciar erro esperado de falha inesperada.

## Onde Esta Aula Entra no Curso

- Modulo: 07. Next.js e Aplicacoes Fullstack no Frontend
- Aula anterior: Loading states e streaming
- Proxima aula: Suspense no contexto do Next.js
- Pre-requisitos: Tratamento de erros, rotas, layouts, chamadas de API e estados de UI.

## Ideia Central

Aplicacoes reais falham. Uma API pode cair, uma permissao pode faltar, um dado pode vir invalido. Error boundaries permitem mostrar uma interface controlada quando uma parte da aplicacao quebra.

No App Router, um arquivo `error.tsx` cria uma fronteira de erro para um segmento de rota.

## Topicos da Aula

- Falhas em UI.
- Arquivo `error.tsx`.
- Error boundary por segmento.
- Botao de tentar novamente.
- Erros esperados vs inesperados.
- Relacao com loading e empty state.

## Exemplo Pratico

```tsx
"use client";

export default function Error({ reset }: { reset: () => void }) {
  return (
    <div>
      <p>Nao foi possivel carregar esta area.</p>
      <button onClick={() => reset()}>Tentar novamente</button>
    </div>
  );
}
```

## Analogias e Intuicoes

- Error boundary e um disjuntor: limita o estrago quando uma parte falha.
- Uma boa mensagem de erro ajuda o usuario a decidir o proximo passo.

## Erros Comuns

- Deixar erro quebrar a tela inteira.
- Mostrar mensagem tecnica demais para o usuario.
- Tratar erro esperado como excecao inesperada.
- Nao oferecer caminho de recuperacao.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- O que `error.tsx` captura?
- Por que ele e um Client Component?
- Quando oferecer tentar novamente?
- Qual a diferenca entre erro, loading e vazio?
- Que mensagens ajudam o usuario?

## Pratica Recomendada

- Crie uma rota que simula erro ao carregar dados.
- Adicione `error.tsx`.
- Escreva uma mensagem clara e um botao de recuperacao.
