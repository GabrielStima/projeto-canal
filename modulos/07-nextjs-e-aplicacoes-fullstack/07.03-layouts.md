## Ideia Central

Layouts definem partes da interface que envolvem paginas. No Next.js, um layout recebe `children` e pode ser compartilhado por uma rota e suas rotas filhas.

Isso permite manter cabecalho, navegacao, sidebar ou estrutura visual sem repetir esse codigo em cada pagina.

## Topicos da Aula

## Exemplo Pratico

```tsx
export default function Layout({ children }: { children: React.ReactNode }) {
  return (
    <>
      <nav>Menu</nav>
      <main>{children}</main>
    </>
  );
}
```

Use um layout para manter uma navegacao fixa nas paginas do painel de estudos.

## Analogias e Intuicoes

- Layout e a moldura persistente; pagina e o quadro que muda.
- Um layout de rota funciona como um componente pai compartilhado.

## Erros Comuns

- Repetir a mesma navegacao em varias paginas.
- Colocar conteudo especifico de pagina no layout.
- Esquecer que layouts tambem seguem a arvore de rotas.
- Transformar todo layout em Client Component sem necessidade.
