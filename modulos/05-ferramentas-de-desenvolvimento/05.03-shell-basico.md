# Shell Basico

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar o papel do shell;
- diferenciar terminal de shell;
- usar comandos simples com opcoes e argumentos;
- saber o que estudar em seguida.

## Onde Esta Aula Entra no Curso

- Modulo: Ferramentas de Desenvolvimento
- Aula anterior: Terminal e linha de comando
- Proxima aula: Ambientes de desenvolvimento
- Pre-requisitos: saber abrir o terminal e navegar por pastas.

## Ideia Central

Terminal e a janela. Shell e o programa que interpreta o que voce digita nessa janela. Bash, zsh e PowerShell sao exemplos de shells.

O shell recebe comandos, expande caminhos, le argumentos, executa programas e mostra a saida. Para desenvolvimento web, voce nao precisa virar especialista em shell no inicio, mas precisa conseguir executar tarefas comuns sem medo.

## Topicos da Aula

- Terminal vs shell.
- Prompt, comando, argumento e flag.
- Variaveis temporarias no shell.
- Saida padrao e mensagens de erro.
- Historico de comandos.
- Cuidados ao copiar comandos destrutivos.

## Exemplo Pratico

Compare comandos com e sem argumentos:

```bash
ls
ls -la
cd ..
pwd
```

Depois use o historico do shell para repetir um comando anterior.

## Analogias e Intuicoes

- Terminal e o balcao; shell e a pessoa que entende o pedido e chama a ferramenta certa.
- Flags sao ajustes finos do comando, como escolher modo detalhado ou resumido.

## Erros Comuns

- Achar que todo terminal usa o mesmo shell.
- Rodar comandos destrutivos sem entender o que fazem.
- Confundir flag com argumento.
- Fechar o terminal achando que isso apaga o projeto.

## Checklist de Entendimento

Antes de seguir, a pessoa deve conseguir responder:

- O que e shell?
- Qual a diferenca entre terminal e shell?
- O que sao argumentos e flags?
- Por que comandos podem se comportar diferente em shells diferentes?

## Pratica Recomendada

- Descubra qual shell voce usa.
- Execute `pwd`, `ls`, `ls -la` e compare as saidas.
- Use seta para cima para recuperar comandos anteriores.
- Escreva uma explicacao curta de terminal vs shell.
