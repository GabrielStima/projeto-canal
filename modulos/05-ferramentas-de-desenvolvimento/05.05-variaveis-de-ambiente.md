# Variaveis de Ambiente

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar o que sao variaveis de ambiente;
- reconhecer quando uma configuracao nao deve ficar fixa no codigo;
- diferenciar `.env` de `.env.example`;
- saber o que estudar em seguida.

## Onde Esta Aula Entra no Curso

- Modulo: Ferramentas de Desenvolvimento
- Aula anterior: Ambientes de desenvolvimento
- Proxima aula: Versionamento semantico
- Pre-requisitos: terminal, shell basico e ideia de ambiente local.

## Ideia Central

Variaveis de ambiente sao valores fornecidos ao programa pelo ambiente onde ele roda. Elas servem para configurar comportamento sem alterar o codigo.

Exemplos comuns: porta do servidor, URL de API, modo de execucao, chaves de servico e credenciais. Valores sensiveis nao devem ser publicados no repositorio.

## Topicos da Aula

- Configuracao vs codigo.
- Variaveis temporarias no terminal.
- Arquivo `.env`.
- Arquivo `.env.example`.
- Segredos e cuidado com Git.
- Diferenca entre desenvolvimento e producao.

## Exemplo Pratico

Um projeto pode ter um arquivo `.env.example` assim:

```env
APP_PORT=3000
API_URL=http://localhost:3000
```

Cada pessoa copia esse arquivo para `.env` e preenche os valores reais do proprio ambiente.

## Analogias e Intuicoes

- Codigo e a receita; variavel de ambiente e o ingrediente escolhido para aquele preparo.
- `.env.example` e uma lista de campos esperados, nao o cofre dos valores reais.

## Erros Comuns

- Commitar `.env` com credenciais.
- Espalhar URLs fixas pelo codigo.
- Nao documentar quais variaveis o projeto espera.
- Usar o mesmo valor sensivel em desenvolvimento e producao.

## Checklist de Entendimento

Antes de seguir, a pessoa deve conseguir responder:

- O que e uma variavel de ambiente?
- Por que nao colocar segredo direto no codigo?
- Para que serve `.env.example`?
- Que tipo de valor pode variar entre ambientes?

## Pratica Recomendada

- Crie um `.env.example` para um projeto ficticio.
- Liste tres variaveis que fariam sentido nesse projeto.
- Escreva no README como criar o `.env`.
- Explique por que o `.env` real nao deve ser versionado.
