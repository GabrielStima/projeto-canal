# Exercício 04 — Scripts de Qualidade do PetCare OS

## O que você já sabe

Você estudou que linters analisam código procurando problemas e padrões, formatters cuidam da aparência e scripts no `package.json` ajudam o time a executar tarefas com comandos previsíveis.

## Cenário

O PetCare OS usa TypeScript. A equipe quer padronizar a qualidade do código com três comandos:

- `npm run check` — verifica tipos com TypeScript.
- `npm run lint` — executa o ESLint no projeto.
- `npm run format` — executa o Prettier para formatar os arquivos.

## Tarefa

Escreva a seção `scripts` de um `package.json` que exponha esses três comandos. Use nomes de ferramentas comuns do ecossistema JavaScript/TypeScript. Depois, explique em uma frase o papel de cada script.

Você não precisa escrever o `package.json` completo, apenas a seção `scripts`.

## Critérios de Correção

- Três scripts com os nomes `check`, `lint` e `format`.
- Comandos que façam sentido para TypeScript, ESLint e Prettier.
- Explicação curta do papel de cada script.
- Nenhum comando que exija configuração avançada que não foi ensinada.
