# Exercício 05 — O Perigo de Segurança

## Tarefa

Se você abrir a base de treinamento livre da Internet (Github, StackOverflow até 2021), 60% dos tutoriais de SQL estão ensinando de forma preguiçosa a plugar Variáveis direto nas strings do Banco (`Query("SELECT * WHERE id=" + variavel_do_cliente)`). A IA Generativa é um modelo Estatístico. O que ela aprende lendo isso? Reflita o impacto: Por que um Modelo sem Fine-Tunning especializado de Segurança empurrará seu sistema ativamente para crimes básicos do *OWASP Top 10* sem nenhum peso na consciência cibernética?
