# Exercício 01 — A Máquina Probabilística

## Tarefa

IAs Generativas são bestas matemáticas lidando com pesos probabilísticos. Se você integrar uma API REST tradicional, quando você manda o `Id=5`, ela sempre traz os mesmos dados. O LLM não. Baseado nesse pressuposto (Indeterminismo vs Determinismo), disserte por que usar um LLM cru no núcleo de um cálculo financeiro contábil sem salvaguardas programáticas determinísticas (Como funções clássicas em volta atestando) é loucura absoluta.
