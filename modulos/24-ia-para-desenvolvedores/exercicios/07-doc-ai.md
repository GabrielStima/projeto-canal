# Exercício 07 — Documentação Instantânea

## Tarefa

Faça uma prova de fogo de IA gerando Alucinações úteis. Peça para o ChatGPT (Ou similar) ler as 10 linhas deste EndPoint (Imagine ou cole um endpoint que busca "Relatórios de Vendas"). Mande gerar uma doc no estilo "Swagger/OpenAPI". A IA inevitavelmente vai "Alucinar/Gessar" campos de Resposta no JSON de saída porque as 10 linhas da API não informam a Interface exata. Analise as invenções (Alucinações Benignas) e critique por que um Desenvolvedor Pleno NUNCA aceita um JSDoc na confiança.

```text

Critérios de correção:
```
