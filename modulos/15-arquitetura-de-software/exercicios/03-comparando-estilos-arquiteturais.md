# Exercício 03 — Comparando Estilos Arquiteturais

## O que você já tem

Seu documento de revisão contém forças arquiteturais, componentes, limites e a direção desejada das dependências.

## Tarefa

Compare quatro alternativas para organizar o fluxo escolhido:

- MVC;
- arquitetura em camadas;
- monólito modular;
- princípios de Clean Architecture combinados com limites de domínio.

Para cada alternativa, registre:

1. como os componentes seriam organizados;
2. qual problema ela resolveria bem;
3. qual custo ou risco adicionaria;
4. se combina com as restrições atuais do projeto.

Ao final, escolha uma arquitetura-base para o PetCare OS. Você pode combinar ideias, mas deve explicar o papel de cada uma e evitar camadas sem responsabilidade real.

## O que você vai produzir

Uma comparação argumentada e uma hipótese de arquitetura-base para validar nos exercícios seguintes.
