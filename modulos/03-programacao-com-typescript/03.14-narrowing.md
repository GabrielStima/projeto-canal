## Ideia Central

Narrowing e o processo de reduzir um tipo amplo para um tipo mais especifico com base em uma checagem. Se uma variavel pode ser `string` ou `number`, uma condicao com `typeof` permite tratar cada caso com seguranca.

Isso e essencial porque dados reais frequentemente podem assumir mais de um formato.

## Exemplo Pratico

```typescript
function imprimirId(id: string | number) {
  if (typeof id === "string") {
    console.log(id.toUpperCase());
    return;
  }

  console.log(id.toFixed(0));
}
```

## Analogias e Intuicoes

- Narrowing e perguntar antes de usar.
- Union type representa possibilidades.
- A checagem elimina possibilidades ate sobrar um caso seguro.

## Erros Comuns

- Acessar metodo de `string` sem checar se o valor e `string`.
- Usar `as` para forcar tipo em vez de checar.
- Criar unions grandes demais sem discriminador claro.
- Esquecer de tratar todos os casos.
- Confundir narrowing com conversao de valor.
