# JavaScript Moderno

## Objetivo

Ao final desta aula, você deve ser capaz de:

- reconhecer recursos modernos usados em código JavaScript real;
- usar `let`, `const`, template strings, arrow functions e destructuring;
- entender por que JavaScript moderno aparece antes de TypeScript;
- ler exemplos simples sem depender de sintaxe antiga.

## Onde Esta Aula Entra na Formação

- Módulo: Programação com TypeScript
- Aula anterior: Programação com TypeScript
- Próxima aula: Sintaxe, valores e variáveis
- Pré-requisitos: JavaScript no browser, DOM, eventos e projeto prático do módulo Frontend Essencial

## Ideia Central

JavaScript moderno é o JavaScript escrito com recursos que se tornaram padrão nos projetos atuais. Antes de falar de tipos, precisamos garantir que a sintaxe cotidiana da linguagem não pareca estranha.

TypeScript usa a mesma base de JavaScript. Se você entende como o JavaScript moderno expressa valores, funções, objetos e arrays, TypeScript vira uma camada a mais de clareza, não uma segunda linguagem desconectada.

## Exemplo Prático

```javascript
const usuario = {
  nome: "Ana",
  area: "frontend",
};

const mensagem = `Olá, ${usuario.nome}. Você está estudando ${usuario.area}.`;

const criarResumo = ({ nome, area }) => {
  return `${nome} - ${area}`;
};

console.log(mensagem);
console.log(criarResumo(usuario));
```

Variação com dados do Portal do Tutor:

```javascript
const tutor = {
  nome: "Carlos",
  pet: "Thor",
  clinica: "PetCare OS - Unidade Centro",
};

const resumo = ({ nome, pet, clinica }) => {
  return `${nome} acompanha ${pet} na ${clinica}.`;
};

console.log(resumo(tutor));
```

## Analogias e Intuições

- JavaScript moderno é o vocabulário atual da linguagem.
- TypeScript entra depois como revisão automatica do que você escreveu.
- Sintaxe nova não significa lógica nova em todos os casos.

## Erros Comuns

- Achar que todo recurso moderno e TypeScript.
- Usar `let` para valores que nunca mudam.
- Usar arrow function sem entender parâmetros e retorno.
- Copiar destructuring sem saber de onde os valores estão vindo.
- Confundir sintaxe moderna com suporte garantido em qualquer ambiente antigo.

## Checklist de Entendimento

Antes de seguir, você deve conseguir responder:

- Quando usar `const`?
- O que uma template string facilita?
- O que uma arrow function muda na escrita?
- O que destructuring extrai?
- Por que JavaScript moderno vem antes de TypeScript?

## Prática Recomendada

- Reescreva um pequeno código antigo usando `const`, template strings e arrow functions.
- Crie um objeto `produto` e extraia campos com destructuring.
- Crie uma função que recebe um objeto e retorna uma mensagem.
- Use spread para criar uma copia alterada de um objeto.
- Crie um objeto representando um resumo de atendimento (tutor, pet, status) e use template string para montar uma mensagem para o portal.
