# Funções

## Objetivo

Ao final desta aula, você deve ser capaz de:

- declarar e chamar funções;
- usar parâmetros e retorno;
- diferenciar efeito colateral e valor retornado;
- preparar a base para tipos em funções no TypeScript.

## Ideia Central

Funções permitem dar nome a uma operação. Em vez de repetir o mesmo bloco de código em vários lugares, criamos uma função que recebe entradas, executa uma lógica e pode devolver uma saída.

Uma boa função torna o código mais fácil de ler, testar e reutilizar.

## Exemplo Prático

```javascript
function calcularTotal(preco, quantidade) {
  return preco * quantidade;
}

const formatarResumo = (nome, total) => {
  return `${nome}: R$ ${total}`;
};

const total = calcularTotal(25, 3);
console.log(formatarResumo("Camiseta", total));
```

## Analogias e Intuições

- Funcao é uma receita com nome.
- Parametros são os ingredientes que variam.
- Retorno é o resultado produzido pela receita.

## Erros Comuns

- Criar funções que fazem muitas coisas ao mesmo tempo.
- Confundir `return` com `console.log`.
- Depender de variáveis externas sem necessidade.
- Nomear funções com substantivos em vez de ações.
- Retornar tipos diferentes sem uma razao clara.
