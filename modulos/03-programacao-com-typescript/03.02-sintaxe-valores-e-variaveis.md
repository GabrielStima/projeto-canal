# Sintaxe, Valores e Variaveis

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar o que e sintaxe;
- diferenciar valor e variavel;
- declarar variaveis com `const` e `let`;
- reconhecer atribuicao, leitura e reatribuicao;
- evitar erros simples de nome, escopo e inicializacao.

## Onde Esta Aula Entra no Curso

- Modulo: Programacao com TypeScript
- Aula anterior: JavaScript moderno
- Proxima aula: Tipos primitivos e estruturas basicas
- Pre-requisitos: JavaScript moderno em nivel introdutorio

## Ideia Central

Todo programa manipula valores. Uma variavel e um nome que aponta para um valor para que possamos reutiliza-lo, compara-lo ou transforma-lo.

Sintaxe e o conjunto de regras de escrita que a linguagem entende. Quando a sintaxe esta errada, o codigo nem chega a expressar corretamente a ideia.

## Topicos da Aula

- O que e sintaxe
- Literais e valores
- Declaracao com `const`
- Declaracao com `let`
- Atribuicao e reatribuicao
- Nomes de variaveis
- Escopo em blocos

## Exemplo Pratico

```javascript
const nome = "Ana";
let pontos = 0;

pontos = pontos + 10;

if (pontos >= 10) {
  const mensagem = `${nome} passou de fase.`;
  console.log(mensagem);
}
```

## Analogias e Intuicoes

- Valor e a informacao; variavel e o nome que usamos para encontra-la.
- `const` protege a ligacao entre nome e valor.
- `let` sinaliza que o valor pode mudar ao longo do fluxo.

## Erros Comuns

- Criar nomes vagos como `x`, `data` ou `temp` sem necessidade.
- Reatribuir uma variavel declarada com `const`.
- Usar uma variavel antes de declarar.
- Confundir mudar um objeto com reatribuir a variavel.
- Declarar variaveis longe demais do ponto onde sao usadas.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- O que e um valor?
- O que e uma variavel?
- Qual a diferenca entre `const` e `let`?
- O que significa reatribuir?
- O que e escopo de bloco?

## Pratica Recomendada

- Crie variaveis para representar nome, idade e cidade.
- Crie uma mensagem usando esses valores.
- Use `let` para simular uma pontuacao que aumenta.
- Teste um exemplo que gera erro ao tentar reatribuir uma `const`.
