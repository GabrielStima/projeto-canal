# Sintaxe, Valores e Variáveis

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar o que é sintaxe;
- diferenciar valor e variável;
- declarar variáveis com `const` e `let`;
- reconhecer atribuição, leitura e reatribuição;
- evitar erros simples de nome, escopo e inicializacao.

## Onde Esta Aula Entra na Formação

- Módulo: Programação com TypeScript
- Aula anterior: JavaScript moderno
- Próxima aula: Tipos primitivos e estruturas básicas
- Pré-requisitos: JavaScript moderno em nível introdutório

## Ideia Central

Todo programa manipula valores. Uma variável é um nome que aponta para um valor para que possamos reutiliza-lo, compara-lo ou transforma-lo.

Sintaxe é o conjunto de regras de escrita que a linguagem entende. Quando a sintaxe está errada, o código nem chega a expressar corretamente a ideia.

## Tópicos da Aula

- O que é sintaxe
- Literais e valores
- Declaracao com `const`
- Declaracao com `let`
- Atribuicao e reatribuição
- Nomes de variáveis
- Escopo em blocos

## Exemplo Prático

```javascript
const nome = "Ana";
let pontos = 0;

pontos = pontos + 10;

if (pontos >= 10) {
  const mensagem = `${nome} passou de fase.`;
  console.log(mensagem);
}
```

## Analogias e Intuições

- Valor é a informação; variável é o nome que usamos para encontra-la.
- `const` protege a ligacao entre nome e valor.
- `let` sinaliza que o valor pode mudar ao longo do fluxo.

## Erros Comuns

- Criar nomes vagos como `x`, `data` ou `temp` sem necessidade.
- Reatribuir uma variável declarada com `const`.
- Usar uma variável antes de declarar.
- Confundir mudar um objeto com reatribuir a variável.
- Declarar variáveis longe demais do ponto onde são usadas.

## Checklist de Entendimento

Antes de seguir, você deve conseguir responder:

- O que é um valor?
- O que é uma variável?
- Qual a diferença entre `const` e `let`?
- O que significa reatribuir?
- O que é escopo de bloco?

## Prática Recomendada

- Crie variáveis para representar nome, idade e cidade.
- Crie uma mensagem usando esses valores.
- Use `let` para simular uma pontuação que aumenta.
- Teste um exemplo que gera erro ao tentar reatribuir uma `const`.
