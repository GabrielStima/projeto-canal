# Módulos

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar por que dividir código em arquivos;
- usar `export` e `import`;
- diferenciar export nomeado e export default em nível introdutório;
- organizar funções e tipos em módulos simples;
- preparar a base para projetos maiores.

## Onde Esta Aula Entra no Curso

- Módulo: Programação com TypeScript
- Aula anterior: Type aliases e interfaces
- Próxima aula: Classes e POO básica

## Ideia Central

Módulos permitem dividir o código em partes menores. Em vez de colocar tudo em um arquivo, exportamos o que precisa ser usado fora e importamos apenas o que outro arquivo precisa.

Isso melhora organização, leitura e manutenção, principalmente quando o projeto cresce.

## Tópicos da Aula

- Por que modularizar
- `export`
- `import`
- Exports nomeados
- Export default em nível introdutório
- Organização de tipos
- Cuidados com dependencias circulares

## Exemplo Prático

```typescript
// tarefa.ts
export type Tarefa = {
  titulo: string;
  concluida: boolean;
};

export function criarTarefa(titulo: string): Tarefa {
  return {
    titulo,
    concluida: false,
  };
}
```

```typescript
// app.ts
import { criarTarefa } from "./tarefa";

const tarefa = criarTarefa("Estudar TypeScript");
console.log(tarefa);
```

## Analogias e Intuições

- Módulo é uma gaveta organizada do projeto.
- Exportar e deixar algo disponível para fora.
- Importar e pedir algo de outro arquivo.

## Erros Comuns

- Exportar tudo sem critério.
- Criar arquivos pequenos demais sem motivo.
- Criar arquivos grandes que fazem muitas coisas.
- Misturar regra, exemplo e execução no mesmo lugar sempre.
- Criar dependencia circular entre arquivos.

## Checklist de Entendimento

Antes de seguir, você deve conseguir responder:

- Por que dividir código em módulos?
- O que `export` faz?
- O que `import` faz?
- Qual a diferença entre exportar uma função é um tipo?
- Que problema dependencia circular pode causar?

## Prática Recomendada

- Separe um tipo `Tarefa` em um arquivo.
- Separe uma função `criarTarefa` no mesmo módulo.
- Importe a função em outro arquivo.
- Experimente renomear um export nomeado e ajustar o import.
