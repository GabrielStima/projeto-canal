# Classes e POO Basica

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar o que e uma classe em JavaScript/TypeScript;
- criar instancias com `new`;
- usar propriedades e metodos;
- entender encapsulamento em nivel introdutorio;
- reconhecer quando classe ajuda e quando funcao simples basta.

## Ideia Central

Classes permitem criar objetos a partir de um molde com dados e comportamentos relacionados. Programacao orientada a objetos organiza codigo em torno de entidades, responsabilidades e interacoes.

Neste ponto do curso, o objetivo e entender o basico sem transformar tudo em classe. Muitas vezes uma funcao e um objeto simples resolvem melhor.

## Exemplo Pratico

```typescript
class Contador {
  private valor = 0;

  incrementar() {
    this.valor += 1;
  }

  obterValor() {
    return this.valor;
  }
}

const contador = new Contador();
contador.incrementar();
console.log(contador.obterValor());
```

## Analogias e Intuicoes

- Classe e um molde.
- Instancia e um objeto criado a partir do molde.
- Metodo e uma acao que pertence ao objeto.

## Erros Comuns

- Criar classe para qualquer agrupamento de funcoes.
- Usar heranca cedo demais.
- Misturar muitas responsabilidades na mesma classe.
- Acessar estado interno sem criterio.
- Confundir classe com tipo, embora uma classe tambem possa ser usada como tipo.
