# Classes e POO Básica

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar o que é uma classe em JavaScript/TypeScript;
- criar instancias com `new`;
- usar propriedades e métodos;
- entender encapsulamento em nível introdutório;
- reconhecer quando classe ajuda e quando função simples basta.

## Ideia Central

Classes permitem criar objetos a partir de um molde com dados e comportamentos relacionados. Programação orientada a objetos organiza código em torno de entidades, responsabilidades e interações.

Neste ponto do curso, o objetivo e entender o básico sem transformar tudo em classe. Muitas vezes uma função é um objeto simples resolvem melhor.

## Tópicos da Aula

- `private` e `public` em nível introdutório
- Heranca apenas como visão geral

## Exemplo Prático

```typescript
class Contador {
  private valor = 0;

  incrementar() {
    this.valor += 1;
  }

  obterValor() {
    return this.valor;
  }
}

const contador = new Contador();
contador.incrementar();
console.log(contador.obterValor());
```

## Analogias e Intuições

- Classe é um molde.
- Instancia é um objeto criado a partir do molde.
- Metodo é uma acao que pertence ao objeto.

## Erros Comuns

- Criar classe para qualquer agrupamento de funções.
- Usar heranca cedo demais.
- Misturar muitas responsabilidades na mesma classe.
- Acessar estado interno sem critério.
- Confundir classe com tipo, embora uma classe também possa ser usada como tipo.
