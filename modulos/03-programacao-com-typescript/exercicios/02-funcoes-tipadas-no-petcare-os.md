# Exercício 02 — Funções Tipadas no PetCare OS

## Objetivo

Praticar anotações de tipos em funções, parâmetros e retornos, usando o contexto do PetCare OS.

## O que você já sabe

- Declarar variáveis com tipos em TypeScript.
- Criar funções em JavaScript.
- Usar `string`, `number`, `boolean`, arrays e objetos.

## Tarefa atual

Você vai criar funções tipadas para manipular dados simples do PetCare OS.

## Exercício

1. Crie uma função `calcularTotalDoOrcamento(itens)` que receba um array de objetos com `descricao: string` e `valor: number` e retorne o valor total como `number`.
2. Crie uma função `formatarStatus(status)` que receba uma string e retorne uma string formatada para exibição (por exemplo, `"Em Triagem"` vira `"Em triagem"`).
3. Crie uma função `estaEmAtendimento(status)` que receba uma string e retorne `boolean` indicando se o status é diferente de `"Finalizado"`.
4. Crie uma função `obterIniciaisDoTutor(nome)` que receba uma string, divida em palavras e retorne as iniciais em maiúsculas.
5. Anote os tipos de todos os parâmetros e retornos.
6. Chame cada função com exemplos do PetCare OS e exiba os resultados.

## Critérios de conclusão

- Todos os parâmetros têm tipos explícitos.
- Todos os retornos têm tipos explícitos.
- As funções funcionam corretamente para os exemplos do PetCare OS.
- Não há uso de `any`.

## Como este trabalho continuará

Funções tipadas como essas serão usadas no projeto final deste módulo e reaparecerão nos módulos de frontend moderno, Next.js e backend, quando você manipular dados do PetCare OS de forma segura.
