# Estruturas de Controle

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- usar condicionais para tomar decisoes;
- usar loops para repetir operacoes;
- entender como o fluxo do programa muda;
- evitar repeticoes desnecessarias;
- preparar exemplos para funcoes e tratamento de erros.

## Onde Esta Aula Entra no Curso

- Modulo: Programacao com TypeScript
- Aula anterior: Tipos primitivos e estruturas basicas
- Proxima aula: Funcoes
- Pre-requisitos: valores, variaveis, tipos primitivos, arrays e objetos simples

## Ideia Central

Um programa raramente executa tudo de cima para baixo sem escolhas. Estruturas de controle permitem decidir quando algo deve acontecer e quantas vezes uma operacao deve se repetir.

Condicionais lidam com possibilidades. Loops lidam com repeticao.

## Topicos da Aula

- `if`, `else if` e `else`
- Comparacoes
- Operadores logicos
- `switch`
- `for`
- `for...of`
- `while`

## Exemplo Pratico

```javascript
const tarefas = ["estudar", "praticar", "revisar"];

for (const tarefa of tarefas) {
  if (tarefa === "praticar") {
    console.log("Reserve mais tempo para praticar.");
  } else {
    console.log(`Tarefa: ${tarefa}`);
  }
}
```

## Analogias e Intuicoes

- Condicional e uma bifurcacao no caminho.
- Loop e uma rotina repetida enquanto existe trabalho a fazer.
- Operadores logicos combinam criterios de decisao.

## Erros Comuns

- Criar condicionais que nunca sao atingidas.
- Confundir `=` com `===`.
- Esquecer de atualizar a condicao de um `while`.
- Usar loop quando um metodo de array seria mais legivel depois.
- Escrever muitas condicoes aninhadas sem necessidade.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- Quando usar `if`?
- Quando usar `switch`?
- Qual a diferenca entre `for` e `for...of`?
- O que pode causar um loop infinito?
- Como operadores logicos mudam uma condicao?

## Pratica Recomendada

- Crie uma lista de notas e mostre quais foram aprovadas.
- Escreva uma condicao com dois criterios.
- Use `switch` para tratar status de tarefa.
- Crie um loop que percorre uma lista de objetos.
