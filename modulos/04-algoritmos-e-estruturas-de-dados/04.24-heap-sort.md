## Ideia Central

Heap Sort usa um heap para transformar o maior item em algo fácil de acessar. Em uma estratégia com max-heap, o maior valor fica no topo; então ele pode ser colocado no final da área ordenada.

Depois de remover o topo, o heap precisa ser reorganizado. Repetindo esse processo, o array fica ordenado.

## Tópicos da Aula

- Construir um heap.
- Remover ou trocar o topo.
- Restaurar a propriedade de heap.
- Complexidade `O(n log n)`.

## Exemplo Prático

```ts
function swap(values: number[], first: number, second: number): void {
  const temp = values[first];
  values[first] = values[second];
  values[second] = temp;
}
```

Heap Sort completo exige operações auxiliares como `heapify`. Nesta aula, o foco e entender o fluxo: criar heap, mover maior para o final e restaurar heap.

## Analogias e Intuições

- Sempre pegar o item de maior prioridade.
- Colocar o maior item na posição final correta.
- Reorganizar a fila de prioridade após cada remoção.

## Erros Comuns

- Tentar entender Heap Sort sem entender heap.
- Confundir propriedade de heap com array totalmente ordenado.
- Esquecer de reduzir a área do heap após cada troca.
- Perder índices de pai e filhos.
