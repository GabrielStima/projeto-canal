# Arvore binaria

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar o que torna uma arvore binaria;
- modelar nos com filho esquerdo e direito;
- percorrer uma arvore binaria em nivel introdutorio;
- preparar a arvore de busca binaria.

## Onde Esta Aula Entra no Curso

- Modulo: 04. Algoritmos e Estruturas de Dados
- Aula anterior: Arvores
- Proxima aula: Arvore de busca binaria
- Pre-requisitos: arvores, objetos, tipos genericos e recursao.

## Ideia Central

Arvore binaria e uma arvore em que cada no tem no maximo dois filhos: esquerdo e direito. Essa restricao cria uma estrutura simples para explicar percursos, buscas e outras arvores especializadas.

Ela ainda nao define uma regra de ordenacao. Isso vem na arvore de busca binaria.

## Topicos da Aula

- Filho esquerdo e filho direito.
- No vazio.
- Percurso recursivo.
- Diferenca entre arvore binaria e árvore de busca binária (BST).

## Exemplo Pratico

```ts
type BinaryNode<T> = {
  value: T;
  left: BinaryNode<T> | null;
  right: BinaryNode<T> | null;
};

const root: BinaryNode<string> = {
  value: "A",
  left: { value: "B", left: null, right: null },
  right: { value: "C", left: null, right: null },
};
```

## Analogias e Intuicoes

- Uma decisao que sempre se divide em duas possibilidades.
- Uma arvore genealogica simplificada.
- Uma estrutura com lado esquerdo e direito.

## Erros Comuns

- Achar que esquerda sempre significa menor.
- Esquecer que filhos podem ser `null`.
- Confundir arvore binaria com arvore balanceada.
- Ignorar o caso de arvore vazia.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- Quantos filhos um no pode ter?
- O que diferencia arvore binaria de arvore comum?
- Uma arvore binaria precisa estar ordenada?
- Como a recursao percorre os nos?

## Pratica Recomendada

- Modele uma arvore binaria com numeros.
- Escreva uma funcao que conte nos.
- Escreva uma funcao que imprima os valores em pre-ordem.
