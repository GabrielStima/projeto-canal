## Ideia Central

Array é uma coleção ordenada de valores. Em JavaScript, arrays são flexíveis e muito usados, mas essa facilidade pode esconder custos diferentes para cada operação.

Acessar por índice é direto. Percorrer todos os itens exige olhar um por um. Inserir ou remover no meio pode exigir deslocar vários elementos.

## Exemplo Prático

```ts
type StockItem = {
  id: number;
  name: string;
  quantity: number;
};

const stock: StockItem[] = [
  { id: 1, name: "Anti-inflamatório", quantity: 30 },
  { id: 2, name: "Soro fisiológico", quantity: 15 },
];

function findItemById(id: number): StockItem | undefined {
  return stock.find((item) => item.id === id);
}
```

Entrada: um `id`. Processamento: percorrer o array até encontrar o item. Saída: o item ou `undefined`.

## Analogias e Intuições

- Uma prateleira numerada.
- Uma playlist em ordem.
- Uma tabela simples onde cada linha tem uma posição.

## Erros Comuns

- Achar que todo acesso a array tem o mesmo custo.
- Usar array para busca frequente por chave sem avaliar alternativas.
- Alterar o array enquanto percorre sem cuidado.
- Confundir índice com identificador do dado.
