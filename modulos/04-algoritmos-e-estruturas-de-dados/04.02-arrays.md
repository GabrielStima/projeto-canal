## Ideia Central

Array e uma colecao ordenada de valores. Em JavaScript, arrays sao flexiveis e muito usados, mas essa facilidade pode esconder custos diferentes para cada operacao.

Acessar por indice e direto. Percorrer todos os itens exige olhar um por um. Inserir ou remover no meio pode exigir deslocar varios elementos.

## Topicos da Aula

- Quando arrays sao uma boa escolha.

## Exemplo Pratico

```ts
type Product = {
  id: string;
  name: string;
  price: number;
};

const products: Product[] = [
  { id: "p1", name: "Teclado", price: 150 },
  { id: "p2", name: "Mouse", price: 80 },
];

function findProductById(id: string): Product | undefined {
  return products.find((product) => product.id === id);
}
```

Entrada: um `id`. Processamento: percorrer o array ate encontrar o produto. Saida: o produto ou `undefined`.

## Analogias e Intuicoes

- Uma prateleira numerada.
- Uma playlist em ordem.
- Uma tabela simples onde cada linha tem uma posicao.

## Erros Comuns

- Achar que todo acesso a array tem o mesmo custo.
- Usar array para busca frequente por chave sem avaliar alternativas.
- Alterar o array enquanto percorre sem cuidado.
- Confundir indice com identificador do dado.
