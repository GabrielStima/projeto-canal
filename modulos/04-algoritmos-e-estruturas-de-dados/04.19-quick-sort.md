## Ideia Central

Quick Sort escolhe um pivo e reorganiza os itens em torno dele: menores de um lado, maiores do outro. Depois repete o processo nas partes menores.

Ele costuma ser eficiente na pratica, mas a escolha do pivo influencia muito. Em escolhas ruins, pode cair para custo quadratico.

## Topicos da Aula

- Pivo.
- Particionamento.
- Chamadas recursivas.
- Caso medio e pior caso.

## Exemplo Pratico

```ts
function quickSort(values: number[]): number[] {
  if (values.length <= 1) {
    return values;
  }

  const [pivot, ...rest] = values;
  const smaller = rest.filter((value) => value <= pivot);
  const greater = rest.filter((value) => value > pivot);

  return [...quickSort(smaller), pivot, ...quickSort(greater)];
}
```

Esta versao e didatica e usa arrays auxiliares. Implementacoes in-place exigem mais cuidado.

## Analogias e Intuicoes

- Escolher uma referencia e separar menores e maiores.
- Organizar pessoas em torno de uma altura escolhida.
- Resolver subgrupos depois da separacao.

## Erros Comuns

- Escolher pivo sem entender impacto.
- Esquecer caso base.
- Confundir particionamento com merge.
- Achar que Quick Sort e sempre `O(n log n)`.

## Checklist de Entendimento
