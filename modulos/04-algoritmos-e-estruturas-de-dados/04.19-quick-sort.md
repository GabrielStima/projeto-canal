## Ideia Central

Quick Sort escolhe um pivô e reorganiza os itens em torno dele: menores de um lado, maiores do outro. Depois repete o processo nas partes menores.

Ele costuma ser eficiente na prática, mas a escolha do pivô influencia muito. Em escolhas ruins, pode cair para custo quadrático.

## Tópicos da Aula

- Pivo.
- Particionamento.
- Chamadas recursivas.
- Caso médio e pior caso.

## Exemplo Prático

```ts
function quickSort(values: number[]): number[] {
  if (values.length <= 1) {
    return values;
  }

  const [pivot, ...rest] = values;
  const smaller = rest.filter((value) => value <= pivot);
  const greater = rest.filter((value) => value > pivot);

  return [...quickSort(smaller), pivot, ...quickSort(greater)];
}
```

Esta versão é didática e usa arrays auxiliares. Implementacoes in-place exigem mais cuidado.

## Analogias e Intuições

- Escolher uma referência e separar menores e maiores.
- Organizar pessoas em torno de uma altura escolhida.
- Resolver subgrupos depois da separacao.

## Erros Comuns

- Escolher pivô sem entender impacto.
- Esquecer caso base.
- Confundir particionamento com merge.
- Achar que Quick Sort é sempre `O(n log n)`.
