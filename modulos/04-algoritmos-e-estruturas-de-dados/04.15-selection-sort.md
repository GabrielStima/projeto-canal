## Ideia Central

Selection Sort divide o array em uma parte ordenada e outra não ordenada. A cada passo, ele procura o menor item da parte não ordenada e coloca na próxima posição correta.

Ele reduz o número de trocas em relação ao Bubble Sort, mas ainda precisa fazer muitas comparações.

## Tópicos da Aula

- Parte ordenada e parte não ordenada.
- Encontrar o menor valor.
- Trocar com a posição atual.
- Complexidade quadrática.

## Exemplo Prático

```ts
function selectionSort(values: number[]): number[] {
  const result = [...values];

  for (let i = 0; i < result.length; i++) {
    let minIndex = i;

    for (let j = i + 1; j < result.length; j++) {
      if (result[j] < result[minIndex]) {
        minIndex = j;
      }
    }

    const temp = result[i];
    result[i] = result[minIndex];
    result[minIndex] = temp;
  }

  return result;
}
```

## Analogias e Intuições

- Escolher a menor carta e colocar no início.
- Montar uma fila do menor para o maior.
- Separar o que já está ordenado do que ainda falta.

## Erros Comuns

- Trocar dentro do loop interno antes de terminar a busca.
- Não inicializar `minIndex` corretamente.
- Achar que menos trocas elimina o custo quadrático.
- Confundir Selection Sort com Insertion Sort.
