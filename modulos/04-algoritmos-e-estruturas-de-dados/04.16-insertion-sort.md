## Ideia Central

Insertion Sort percorre o array como se estivesse montando uma mao de cartas. Cada novo item e inserido na posicao correta dentro da parte ja ordenada.

Ele ainda pode ter custo quadratico, mas e intuitivo e pode ser eficiente quando a entrada ja esta quase em ordem.

## Topicos da Aula

- Parte ordenada.
- Item atual.
- Deslocar valores maiores.
- Inserir na posicao correta.

## Exemplo Pratico

```ts
function insertionSort(values: number[]): number[] {
  const result = [...values];

  for (let i = 1; i < result.length; i++) {
    const current = result[i];
    let j = i - 1;

    while (j >= 0 && result[j] > current) {
      result[j + 1] = result[j];
      j--;
    }

    result[j + 1] = current;
  }

  return result;
}
```

## Analogias e Intuicoes

- Organizar cartas na mao.
- Inserir um livro na posicao certa da prateleira.
- Manter uma parte sempre ordenada.

## Erros Comuns

- Sobrescrever o item atual antes de guardar em `current`.
- Errar a condicao do `while`.
- Esquecer de inserir `current` ao final.
- Pensar que todo caso sera rapido.
