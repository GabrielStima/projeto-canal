## Ideia Central

Insertion Sort percorre o array como se estivesse montando uma mão de cartas. Cada novo item é inserido na posição correta dentro da parte já ordenada.

Ele ainda pode ter custo quadrático, mas é intuitivo e pode ser eficiente quando a entrada já está quase em ordem.

## Tópicos da Aula

- Parte ordenada.
- Item atual.
- Deslocar valores maiores.
- Inserir na posição correta.

## Exemplo Prático

```ts
function insertionSort(values: number[]): number[] {
  const result = [...values];

  for (let i = 1; i < result.length; i++) {
    const current = result[i];
    let j = i - 1;

    while (j >= 0 && result[j] > current) {
      result[j + 1] = result[j];
      j--;
    }

    result[j + 1] = current;
  }

  return result;
}
```

## Analogias e Intuições

- Organizar cartas na mão.
- Inserir um livro na posição certa da prateleira.
- Manter uma parte sempre ordenada.

## Erros Comuns

- Sobrescrever o item atual antes de guardar em `current`.
- Errar a condicao do `while`.
- Esquecer de inserir `current` ao final.
- Pensar que todo caso sera rápido.
