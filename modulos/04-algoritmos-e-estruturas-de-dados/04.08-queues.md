## Ideia Central

Queue, ou fila, é uma estrutura em que o primeiro item inserido e o primeiro a sair. Esse comportamento é chamado de FIFO: first in, first out.

Filas aparecem em atendimento, processamento de mensagens, tarefas pendentes, impressão, eventos e sistemas distribuídos.

Em uma clínica veterinária, a fila de atendimento simples funciona assim: o primeiro pet que chega é o primeiro a ser atendido. Essa regra muda quando entra a urgência — aí uma fila de prioridade faz mais sentido.

## Tópicos da Aula

## Exemplo Prático

```ts
class Queue<T> {
  private items: T[] = [];

  enqueue(item: T): void {
    this.items.push(item);
  }

  dequeue(): T | undefined {
    return this.items.shift();
  }

  peek(): T | undefined {
    return this.items[0];
  }
}
```

Esta versão é didática. Em arrays grandes, usar `shift` pode ter custo relevante porque desloca itens.

## Analogias e Intuições

- Uma fila de mercado.
- Uma fila de impressao.
- Pedidos aguardando processamento.

## Erros Comuns

- Confundir FIFO com LIFO.
- Usar `shift` sem entender o custo em arrays grandes.
- Não tratar fila vazia.
- Permitir remoção fora da ordem da fila.
