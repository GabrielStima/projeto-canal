## Ideia Central

Queue, ou fila, e uma estrutura em que o primeiro item inserido e o primeiro a sair. Esse comportamento e chamado de FIFO: first in, first out.

Filas aparecem em atendimento, processamento de mensagens, tarefas pendentes, impressao, eventos e sistemas distribuidos.

## Topicos da Aula

- FIFO.
- Operacoes `enqueue`, `dequeue` e `peek`.
- Fila vazia.
- Comparacao com pilha.

## Exemplo Pratico

```ts
class Queue<T> {
  private items: T[] = [];

  enqueue(item: T): void {
    this.items.push(item);
  }

  dequeue(): T | undefined {
    return this.items.shift();
  }

  peek(): T | undefined {
    return this.items[0];
  }
}
```

Esta versao e didatica. Em arrays grandes, usar `shift` pode ter custo relevante porque desloca itens.

## Analogias e Intuicoes

- Uma fila de mercado.
- Uma fila de impressao.
- Pedidos aguardando processamento.

## Erros Comuns

- Confundir FIFO com LIFO.
- Usar `shift` sem entender o custo em arrays grandes.
- Nao tratar fila vazia.
- Permitir remocao fora da ordem da fila.
