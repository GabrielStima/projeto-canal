## Ideia Central

Recursao acontece quando uma funcao resolve um problema chamando a si mesma para uma versao menor do mesmo problema. Para funcionar, ela precisa de um caso base que encerra as chamadas.

Divisao e conquista usa essa ideia para dividir um problema, resolver partes menores e combinar os resultados. Essa estrategia aparece em algoritmos importantes de ordenacao e em varias operacoes com arvores.

## Topicos da Aula

- Dividir, resolver e combinar.

## Exemplo Pratico

```ts
function sum(values: number[]): number {
  if (values.length === 0) {
    return 0;
  }

  const [first, ...rest] = values;
  return first + sum(rest);
}
```

Este exemplo e didatico. Em JavaScript, para arrays grandes, uma versao iterativa pode ser mais segura.

## Analogias e Intuicoes

- Resolver uma escada olhando um degrau por vez.
- Abrir caixas dentro de caixas ate achar uma vazia.
- Dividir uma pilha de provas entre pessoas e juntar os resultados.

## Erros Comuns

- Esquecer o caso base.
- Criar uma chamada recursiva que nao reduz o problema.
- Usar recursao quando um loop seria mais simples.
- Ignorar limite de pilha de chamadas.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- Qual e o caso base?
- O problema fica menor a cada chamada?
- O resultado precisa ser combinado?
- Um loop seria mais claro aqui?

## Pratica Recomendada

- Implemente soma recursiva de um array.
- Implemente contagem regressiva recursiva.
- Descreva, sem codigo, como dividir uma lista ao meio ate sobrar um item.
