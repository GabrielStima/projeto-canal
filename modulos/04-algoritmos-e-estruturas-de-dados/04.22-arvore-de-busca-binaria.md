## Ideia Central

Árvore de busca binária (BST) é uma arvore binaria com regra de ordenacao: valores menores ficam de um lado e valores maiores ficam do outro.

Essa regra permite buscar descartando partes da arvore, parecido com a ideia da busca binaria. O desempenho depende muito do formato da arvore.

## Topicos da Aula

- Regra esquerda/menor e direita/maior.
- Insercao.
- Busca.
- Arvore balanceada vs desbalanceada em alto nivel.

## Exemplo Pratico

```ts
type BinarySearchNode = {
  value: number;
  left: BinarySearchNode | null;
  right: BinarySearchNode | null;
};

function search(root: BinarySearchNode | null, target: number): boolean {
  if (root === null) {
    return false;
  }

  if (root.value === target) {
    return true;
  }

  if (target < root.value) {
    return search(root.left, target);
  }

  return search(root.right, target);
}
```

## Analogias e Intuicoes

- Um jogo de maior ou menor em forma de arvore.
- Um dicionario com caminhos de decisao.
- Um indice que elimina metade idealmente, mas nem sempre perfeitamente.

## Erros Comuns

- Quebrar a regra de ordenacao ao inserir.
- Achar que toda árvore de busca binária (BST) e automaticamente eficiente.
- Ignorar valores duplicados.
- Confundir árvore de busca binária (BST) com heap.

## Checklist de Entendimento

- Como a ideia lembra busca binaria?

## Pratica Recomendada

- Implemente busca em uma árvore de busca binária (BST).
- Implemente insercao de numeros.
- Teste inserir valores ja ordenados e observe o formato resultante.
