## Ideia Central

Stack, ou pilha, e uma estrutura em que o ultimo item inserido e o primeiro a sair. Esse comportamento e chamado de LIFO: last in, first out.

Pilhas aparecem em historico de desfazer, chamadas de funcao, navegacao, validacao de parenteses e processamento de tarefas reversas.

## Topicos da Aula

- LIFO.
- Operacoes `push`, `pop` e `peek`.
- Pilha vazia.
- Casos de uso.

## Exemplo Pratico

```ts
class Stack<T> {
  private items: T[] = [];

  push(item: T): void {
    this.items.push(item);
  }

  pop(): T | undefined {
    return this.items.pop();
  }

  peek(): T | undefined {
    return this.items[this.items.length - 1];
  }
}
```

## Analogias e Intuicoes

- Uma pilha de pratos.
- Desfazer a ultima edicao antes das anteriores.
- Voltar pela ultima pagina visitada.

## Erros Comuns

- Usar pilha quando precisa remover o item mais antigo.
- Nao tratar `pop` em pilha vazia.
- Expor o array interno e quebrar a regra LIFO.
- Confundir `peek` com `pop`.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- Qual item sai primeiro?
- Quais sao as operacoes principais?
- O que acontece quando a pilha esta vazia?
- Onde uma pilha aparece em software real?

## Pratica Recomendada

- Implemente uma pilha generica.
- Use a pilha para validar parenteses simples em uma string.
- Simule um historico de desfazer com `push` e `pop`.
