## Ideia Central

Stack, ou pilha, é uma estrutura em que o último item inserido e o primeiro a sair. Esse comportamento é chamado de LIFO: last in, first out.

Pilhas aparecem em histórico de desfazer, chamadas de função, navegação, validação de parênteses e processamento de tarefas reversas.

## Tópicos da Aula

- LIFO.
- Operações `push`, `pop` e `peek`.
- Pilha vazia.
- Casos de uso.

## Exemplo Prático

```ts
class Stack<T> {
  private items: T[] = [];

  push(item: T): void {
    this.items.push(item);
  }

  pop(): T | undefined {
    return this.items.pop();
  }

  peek(): T | undefined {
    return this.items[this.items.length - 1];
  }
}
```

## Analogias e Intuições

- Uma pilha de pratos.
- Desfazer a última edicao antes das anteriores.
- Voltar pela última pagina visitada.

## Erros Comuns

- Usar pilha quando precisa remover o item mais antigo.
- Não tratar `pop` em pilha vazia.
- Expor o array interno e quebrar a regra LIFO.
- Confundir `peek` com `pop`.
