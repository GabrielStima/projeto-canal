## Ideia Central

Merge Sort divide o array em partes menores até chegar a listas triviais. Depois combina essas partes em ordem usando uma operação chamada merge.

Ele é mais eficiente que os algoritmos quadráticos para entradas grandes, mas normalmente usa memória auxiliar para combinar os arrays.

## Tópicos da Aula

- Dividir ao meio.
- Ordenar partes menores.
- Combinar listas ordenadas.
- Tempo `O(n log n)` e espaço auxiliar.

## Exemplo Prático

```ts
function merge(left: number[], right: number[]): number[] {
  const result: number[] = [];
  let i = 0;
  let j = 0;

  while (i < left.length && j < right.length) {
    if (left[i] <= right[j]) {
      result.push(left[i]);
      i++;
    } else {
      result.push(right[j]);
      j++;
    }
  }

  return result.concat(left.slice(i), right.slice(j));
}
```

## Analogias e Intuições

- Juntar duas filas já ordenadas.
- Dividir uma pilha de cartas e recombinar em ordem.
- Resolver partes pequenas antes do todo.

## Erros Comuns

- Tentar fazer merge de listas que não estão ordenadas.
- Esquecer de adicionar os itens restantes.
- Não definir caso base na recursão.
- Ignorar o custo de memória auxiliar.
