## Ideia Central

Merge Sort divide o array em partes menores ate chegar a listas triviais. Depois combina essas partes em ordem usando uma operacao chamada merge.

Ele e mais eficiente que os algoritmos quadraticos para entradas grandes, mas normalmente usa memoria auxiliar para combinar os arrays.

## Topicos da Aula

- Dividir ao meio.
- Ordenar partes menores.
- Combinar listas ordenadas.
- Tempo `O(n log n)` e espaco auxiliar.

## Exemplo Pratico

```ts
function merge(left: number[], right: number[]): number[] {
  const result: number[] = [];
  let i = 0;
  let j = 0;

  while (i < left.length && j < right.length) {
    if (left[i] <= right[j]) {
      result.push(left[i]);
      i++;
    } else {
      result.push(right[j]);
      j++;
    }
  }

  return result.concat(left.slice(i), right.slice(j));
}
```

## Analogias e Intuicoes

- Juntar duas filas ja ordenadas.
- Dividir uma pilha de cartas e recombinar em ordem.
- Resolver partes pequenas antes do todo.

## Erros Comuns

- Tentar fazer merge de listas que nao estao ordenadas.
- Esquecer de adicionar os itens restantes.
- Nao definir caso base na recursao.
- Ignorar o custo de memoria auxiliar.
