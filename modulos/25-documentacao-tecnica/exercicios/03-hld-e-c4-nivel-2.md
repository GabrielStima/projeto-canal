# Exercício 03 — HLD e C4 Nível 2

## O Que Você Já Possui

Você tem o README do PetCare OS e um ADR registrando uma decisão arquitetural importante. Agora você vai desenhar a arquitetura de alto nível para que diferentes públicos entendam o sistema no zoom certo.

## A Tarefa

Crie `docs/arquitetura/hld.md` com dois diagramas em Mermaid:

1. **Diagrama de contexto (C4 nível 1):** mostre os usuários e sistemas externos que interagem com o PetCare OS. Use caixas para cada ator ou sistema externo e indique o relacionamento.

2. **Diagrama de containers (C4 nível 2):** mostre as aplicações executáveis, bancos, filas, cache e integrações externas. Inclua:
   - frontend web e mobile;
   - API principal e APIs auxiliares, se houver;
   - banco de dados principal;
   - fila e worker;
   - cache;
   - integrações externas;
   - fluxo de uma requisição de leitura e de uma requisição de escrita assíncrona.

Para cada diagrama, adicione:

- título e objetivo;
- público-alvo;
- legenda com os símbolos usados;
- uma descrição textual de duas a quatro frases explicando o que o diagrama responde.

Não inclua classes, funções, interfaces ou detalhes de código nesse nível.

## O Que Você Vai Produzir

Um HLD com C4 nível 1 e nível 2 que será a base para o diagrama detalhado de um fluxo no próximo exercício.
