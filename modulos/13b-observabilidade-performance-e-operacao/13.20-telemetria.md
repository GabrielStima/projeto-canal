# Telemetria

## Objetivo

Ao final desta aula, você deve ser capaz de:

- explicar telemetria em sistemas de software;
- diferenciar emitir sinal de coletar e transportar sinal;
- entender o caminho entre aplicação e análise;
- reconhecer cuidados com volume, custo e privacidade.

## Ideia Central

Telemetria é o conjunto de dados que um sistema envia sobre seu próprio funcionamento. A aplicação emite sinais, esses sinais são coletados, processados, armazenados e consultados para investigação e acompanhamento.

O ponto central aqui não é decorar ferramenta. É entender o fluxo: produzir sinal útil, transportar com contexto, controlar volume, proteger dados sensíveis e permitir consulta posterior.

## Tópicos da Aula

- Sinal emitido.
- Coleta.
- Transporte.
- Armazenamento.
- Consulta.
- Volume, custo e privacidade.

## Exemplo Prático

Fluxo de Dados Operacionais:

A API do PetCare, o Banco Postgres e o App Mobile não mostram dashboards nativos.
A Telemetria consolida tudo.

1. API emite sinal (Log em JSON).
2. O Agente Coletor (Datadog Agent, FluentBit ou Promtail) que está no servidor lê esse JSON silenciosamente.
3. O agente manda o dado via HTTP/gRPC para um Storage Central (Elasticsearch, Loki, Grafana Cloud).
4. O dev analisa no Grafana tudo o que aconteceu na última meia hora.

## Analogias e Intuições

- Telemetria é a frota de caminhões-pipa que busca a água suja (logs não processados) dos bairros (servidores) e leva pra estação de tratamento (Datadog/Grafana) para análise química.

## Erros Comuns

- Coletar tudo sem critério.
- Ignorar custo de volume.
- Enviar dado sensível.
- Perder correlação entre sinais.
- Acreditar que ferramenta resolve pergunta mal definida.

## Checklist de Entendimento

Antes de seguir, você deve conseguir responder:

- Que sinal está sendo enviado?
- Para onde ele vai?
- Como ele será consultado?
- Qual contexto precisa acompanhar?
- Que risco de privacidade existe?

## Prática Recomendada

- Desenhe o diagrama de caixa do fluxo: A API do PetCare gera arquivo `api.log`, o Vector/FluentBit o coleta e envia pro Datadog.
- Avalie o custo financeiro: Por que logar mensagens como 'Entrou no for loop' custa muito caro em volume de dados mensais nos provedores SaaS Cloud?
