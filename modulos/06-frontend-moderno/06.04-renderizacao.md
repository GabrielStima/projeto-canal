# Renderizacao

## Objetivo

Ao final desta aula, voce deve ser capaz de:

- explicar renderizacao em React em nivel pratico;
- entender que componentes descrevem UI para um estado;
- reconhecer por que mudancas de estado causam nova renderizacao;
- evitar confundir renderizacao com manipulacao manual do DOM.

## Ideia Central

Renderizar e calcular como a interface deve aparecer naquele momento. Em React, um componente e executado para produzir uma descricao da UI. Quando estado ou props mudam, React pode renderizar novamente para refletir esses novos dados.

Isso nao significa recriar a pagina inteira manualmente. O programador descreve o resultado esperado; React compara e aplica as mudancas necessarias na interface.

Entender renderizacao ajuda a evitar bugs comuns, como calcular valores no lugar errado, atualizar estado durante renderizacao ou esperar que uma variavel comum preserve memoria entre interacoes.

## Topicos da Aula

- Nova renderizacao por mudanca de estado ou props.

## Exemplo Pratico

```tsx
function Saudacao({ nome }: { nome: string }) {
  if (!nome) {
    return <p>Informe seu nome.</p>;
  }

  return <p>Ola, {nome}!</p>;
}
```

O componente descreve dois possiveis resultados. A UI depende do valor recebido em `nome`.

## Erros Comuns

- Atualizar estado diretamente durante a renderizacao.
- Guardar dados importantes em variaveis comuns dentro do componente.
- Achar que toda nova renderizacao e um bug.
- Fazer calculos caros sem necessidade em toda renderizacao.

## Checklist de Entendimento

Antes de seguir, voce deve conseguir responder:

- O que significa renderizar um componente?
- O que pode causar uma nova renderizacao?
- Por que a UI pode ser vista como funcao dos dados?
- Qual a diferenca entre renderizar e manipular DOM manualmente?
- Como renderizacao condicional funciona?

## Pratica Recomendada

- Crie um componente que renderiza mensagens diferentes para status `carregando`, `erro` e `sucesso`.
- Altere props e observe como a UI mudaria.
- Explique por que uma variavel local comum nao substitui estado.
